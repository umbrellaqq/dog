
# 🐶 Py-apple Dog (ESP32 C语言版机器狗控制系统)

## 📖 项目简介

本项目是一个基于 ESP32 微控制器的 12 自由度（DOF）四足机器人（机器狗）底层控制系统。代码基于 **ESP-IDF** 框架和 **FreeRTOS** 实时操作系统，采用纯 C 语言编写，实现了从底层硬件驱动、运动学逆解计算，到上层 Web 遥控、蓝牙调参的完整全栈闭环。

本系统专为极客和机器人爱好者打造，支持 Web 端的虚拟摇杆遥控、在线舵机标定，以及基于 MPU6050 的姿态解算与闭环控制。

## ⚙️ 技术栈与软件架构

* **核心芯片：** ESP32
* **开发框架：** ESP-IDF (基于 CMake 构建)
* **实时系统：** FreeRTOS (多任务并发与队列通信)
* **硬件外设：** PCA9685 (16路PWM舵机驱动)、MPU6050 (6轴姿态传感器)

系统采用 **“生产者-消费者”** 模式，通过 FreeRTOS 队列（Queue）进行无锁数据传递，确保了运动学解算与硬件驱动的高效同步。

## 📂 核心模块说明 (Components)

本项目将不同功能解耦为独立的模块：

* 🧠 **`main` (主控逻辑)：**
* 统筹整个系统，创建并管理 `servo_smooth`、`gait`、`attitude`、`ik` 和 `main_control` 五个不同优先级的核心任务。
* 实现了基于队列的数据流转（步态数据 + 姿态数据 -> 主控制融合 -> IK逆解 -> 舵机执行）。


* 📐 **`kinematics` (运动学引擎)：**
* 包含了机器狗 12 个自由度的**逆运动学解算 (IK)**。
* 提供机身姿态补偿（Pitch/Roll/Yaw 及三轴平移）算法。


* 🐾 **`gait_planner` (步态生成器)：**
* 内置 Trot（小跑）等步态算法，支持通过摇杆 Y 轴动态调节步长。
* 输出平滑的足端 3D 坐标增量。


* 🌐 **`web` & `wifi` (Web 遥控台)：**
* 开启 ESP32 SoftAP（热点 `PA_Dog_WiFi`）。
* 运行轻量级 HTTP 服务器，提供美观的移动端 HTML 控制界面。
* 支持虚拟摇杆控制、步态切换、高度/姿态实时微调。


* 🔧 **`debug_tool` (标定与调试)：**
* 提供可视化的“90度十字架”舵机物理归中标定界面。
* 结合 `nvs_flash` 模块，实现微调偏移量（Offsets）的掉电记忆与自动加载。


* 📡 **`bluetooth` (原生 BLE 蓝牙)：**
* 基于 NimBLE 协议栈，模拟透传模块服务（FFE0/FFE1）。
* 能够与微信小程序无缝对接，实现运行时的 PID 参数（Kp, Ki, Kd）动态调参。


* 🔌 **底层驱动 (`iic`, `pca9685`, `mpu6050`)：**
* 深度优化的 400kHz I2C 极速通信。
* PCA9685 硬件 PWM 的精准映射与误差扩散补偿算法。
* MPU6050 数据读取与基于互补滤波的 Pitch/Roll 姿态解算。



## ✨ 项目亮点

1. **纯 C 语言硬核实现：** 告别 Arduino 封装的低效，充分压榨 ESP32 的双核性能。
2. **多线程并发安全：** 摇杆数据读取、步态计算、IK 逆解、I2C 发送均在不同的 Task 中运行，互不干扰，保证了运动的丝滑感（高达 50Hz 刷新率）。
3. **软硬件平滑过渡：** 专门编写的 `servo_smooth_task` 确保了机器狗在启动、锁定或急停时，舵机不会产生暴力抽搐，有效保护硬件。
4. **一站式 Web 控制：** 无需额外下载 APP，手机连上机器狗 WiFi 即可直接控制，甚至可以直接在网页上进行每条腿的微调标定并永久保存。
